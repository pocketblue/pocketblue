# Fedora Pocketblue Remix

Pocketblue is a project which provides Fedora Atomic images for mobile devices

### Disclaimer

This is a work-in-progress. During the installation process all data on your device will be wiped.
**Use at your own risk.**

### Supported devices

- [Xiaomi Pad 5](https://pocketblue.github.io/devices/xiaomi-nabu)
- [Xiaomi Pad 6](https://pocketblue.github.io/devices/xiaomi-pipa)
- [OnePlus 6/6T](https://pocketblue.github.io/devices/oneplus-sdm845)
